# 🌐 CRM Imobiliário - Frontend

## 📌 Visão Geral

Interface web do CRM imobiliário integrada à API REST.

### Funcionalidades:
- Login e registro
- Gestão de imóveis
- Gestão de clientes
- Follow-up e pipeline

---

## 🚀 Tecnologias

- HTML
- CSS (Bulma)
- JavaScript
- Fetch API

---

## ⚙️ Pré-requisitos

- Navegador
- Backend rodando

---

## 🔧 Setup

```bash
git clone <repo-frontend>
```

Abrir:
```
index.html
```

---

## 🔐 Autenticação

- Token salvo em:
```
localStorage
```

- Header:
```
Authorization: Bearer TOKEN
```

---

## 🧩 Funcionalidades

### Dashboard
- Listagem de imóveis
- Navegação entre abas

### Imóveis
- CRUD completo
- Visualização detalhada

### Imagens
- Upload (PNG/JPEG)
- URL + base64
- Galeria e viewer

### Clientes
- CRUD completo
- Pipeline (Frio, Morno, Quente)

### Follow-up
- Histórico de notas
- Próxima interação
